HyperOS 系统图标用于导航、操作与内容反馈。文档站提供可搜索的图标预览；设计源与 Figma 组件库保持一致。

## 尺寸

| 场景 | 推荐尺寸 |
|------|----------|
| 列表 leading icon | 24dp |
| 按钮 icon | 20–24dp |
| 导航 icon | 24dp |

画板导出多为 28×28 viewBox，实现时按场景缩放到上表尺寸。

## 使用约定

- **单色图标**：使用 `currentColor`，随文字/主题色变化
- **复制**：预览卡片可复制短名称、稳定 ID（`category.name`）或 SVG 源码
- **工程对齐**：稳定 ID 建议与客户端资源命名对照使用

完整图标库持续扩充；当前页为已入库资产预览。
