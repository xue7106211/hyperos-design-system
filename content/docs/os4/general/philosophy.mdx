# 设计体系与设计系统的定义

### 设计体系（Design Architecture）

- 设计体系是从**理念到系统**的结构化思维框架
- 建立**设计理念**、**设计语言**、**设计原则**与**设计系统**之间的逻辑关系。

### 设计系统（Design System）

- 设计系统是可复用的设计语言与规则体系
- 提供明确的视觉、交互与信息结构标准
- 确保产品在不同平台、设备与场景中都能为用户提供连贯、顺畅的体验

# 全新 HyperOS 4 设计理念与设计语言

<Callout type="idea">
  HyperOS4 继承生命感美学，建立以自然、生命为核心的设计理念，并形成一套风格独特、情感丰富的设计语言。
</Callout>

### HyperOS 生命感设计理念

生命感设计理念由三个核心维度构成：

| 维度 | 主张 |
|------|------|
| **直觉物理 Intuitive Physics** | 「遵循自然物理法则」 |
| **有机空间 Organic Space** | 「超越平面，构建景深」 |
| **灵动响应 Responsive Vitality** | 「与环境共呼吸，随情境而变」 |

### 设计理念的表达

1. **自然法则 Natural Laws**：我们并非仅仅在排列像素，而是在构建一个拥有重力与光影的数字物理世界。
2. **鲜活空间 Living Space**：信息不再是二维平面上的静止印刷品，而是栖息于拥有景深与层级关系的立体空间之中。
3. **流动生态 Fluid Ecology**：系统拒绝僵化静止，它将随着用户情境与环境的变化，保持生生不息的流转与演变。

### 设计语言

Xiaomi HyperOS 的设计语言与设计系统密切相关。

**理性结构 · 感性语言**

设计语言同时涵盖系统的性格（感性表达）与规则（理性秩序）两个方向。

- **设计语言的方向性**决定了**系统性格**的定义
- 系统特征最终会反映在**设计系统**，以及**系统桌面**、**系统应用**等层面
