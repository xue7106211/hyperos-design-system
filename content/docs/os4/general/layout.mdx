## 排版

## 字体家族

移动端默认使用系统字体栈，保证 HyperOS 在各设备上的原生阅读体验。

## 字号层级

| 层级 | 用途 |
|------|------|
| Title | 页面标题 |
| Headline | 区块标题 |
| Body | 正文 |
| Caption | 辅助说明 |

详细 Token 将在 Figma Typography 变量同步后补充。

## 间距

HyperOS 使用 4px 基准网格。间距 Token 定义在 `spacing.scale` 与组件级 `spacing.*` 命名空间下。

## Elevation 层级
