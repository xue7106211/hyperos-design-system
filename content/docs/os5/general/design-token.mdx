## 文件位置

Design Tokens 以 **W3C DTCG** JSON 格式存放在仓库根目录：

```text
tokens/tokens.json
```

## 命名约定

```text
{color|spacing|typography}.{semantic-group}.{property}
```

示例：

- `color.action.primary-bg` — 主要操作按钮背景
- `spacing.button.padding-x` — 按钮水平内边距

## 同步流程（规划）

```text
Figma (Tokens Studio) → GitHub PR → tokens/tokens.json → 文档站 TokenTable
```

Phase 3 将接入 Tokens Studio 与 Style Dictionary 自动构建。

## 色彩体系

HyperOS 色彩体系分为 **品牌色（Brand）**、**语义色（Action / Surface / Text）** 等层级。色彩 Token 采用 W3C DTCG 格式，存放在 `tokens/tokens.json`，可通过 Tokens Studio 从 Figma 同步。

## 使用原则

- 界面中优先使用 **语义 Token**（如 `color.action.primary-bg`），而非直接引用原始色值
- 暗色模式通过 Token 别名切换，避免硬编码 hex
- 确保文本与背景对比度满足 WCAG AA
