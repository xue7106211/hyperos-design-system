Button 用于触发页面内的主要或次要操作。HyperOS Button 支持多种视觉层级，并遵循统一的触控区域与 Token 规范。

**支持平台**：Android · iOS

## 平台实现

<PlatformTabs>
  <PlatformTab platform="android" title="Jetpack Compose">
{`@Composable
fun HyperButton(
    text: String,
    onClick: () -> Unit,
    modifier: Modifier = Modifier,
) {
    Button(
        onClick = onClick,
        modifier = modifier.padding(
            horizontal = HyperTokens.Spacing.Button.paddingX,
            vertical = HyperTokens.Spacing.Button.paddingY,
        ),
        colors = ButtonDefaults.buttonColors(
            containerColor = HyperTokens.Color.Action.primaryBg,
            contentColor = HyperTokens.Color.Action.primaryFg,
        ),
    ) {
        Text(text = text, fontSize = HyperTokens.Typography.Button.label)
    }
}`}
  </PlatformTab>
  <PlatformTab platform="ios" title="SwiftUI">
{`struct HyperButton: View {
    let title: String
    let action: () -> Void

    var body: some View {
        Button(action: action) {
            Text(title)
                .font(.system(size: HyperTokens.Typography.buttonLabel))
                .padding(.horizontal, HyperTokens.Spacing.Button.paddingX)
                .padding(.vertical, HyperTokens.Spacing.Button.paddingY)
        }
        .buttonStyle(.borderedProminent)
        .tint(HyperTokens.Color.Action.primaryBg)
    }
}`}
  </PlatformTab>
</PlatformTabs>

- 为 Icon-only 按钮提供 `contentDescription` / `accessibilityLabel`
- 支持 TalkBack / VoiceOver 朗读按钮角色与状态
